Esomelo App UI Redesign

Started redesigning the UI of the Esomelo application - an app designed for school management. This has been an interesting task because my skills in color selection and discernment are wanting, but it's going well! Working with Figma has opened up new possibilities for creating intuitive and visually appealing interfaces.

Project Overview

  • Analyzing the existing Esomelo school management application
  • Identifying pain points in the current user interface
  • Creating wireframes and mockups in Figma
  • Developing a cohesive color scheme and typography system
  • Focusing on user experience improvements

The Design Process

My approach to redesigning the Esomelo app involved several key phases, each building upon the previous to create a more user-friendly experience.

Step 1: User Research & Analysis

First, I spent time understanding the current application and identifying areas for improvement:

  • Analyzed user flow and navigation patterns
  • Identified cluttered interface elements
  • Noted inconsistent color usage and typography
  • Documented accessibility concerns

Step 2: Wireframing

Created low-fidelity wireframes to establish the basic structure and layout:

  • Simplified navigation menu structure
  • Reorganized dashboard layout for better information hierarchy
  • Designed card-based components for better content organization
  • Planned responsive breakpoints for mobile and tablet views

Step 4: Component Design

Created reusable design components in Figma:

  • Button variations (primary, secondary, danger, success)
  • Form input fields with proper validation states
  • Card components for displaying student and class information
  • Navigation components with active states
  • Modal and dialog designs for actions and confirmations

Key Design Improvements

Information Hierarchy

Used typography scales and spacing to create clear visual hierarchy, making it easier for users to scan and find information quickly.

Consistent Icon Usage

Implemented a consistent icon system using Feather icons, ensuring all interface elements have a unified visual language.

Accessibility Focus

Ensured proper color contrast ratios and included focus states for keyboard navigation, making the app more inclusive.

Mobile-First Approach

Designed responsive layouts that work seamlessly across devices, considering that many users might access the app on mobile devices.

Figma Features I Learned

This project helped me explore several powerful Figma features:

  • Auto Layout: Created responsive components that adapt to content changes
  • Component Variants: Built button and form components with multiple states
  • Color Styles: Established a consistent color system across all designs
  • Typography Styles: Created text styles for headings, body text, and captions
  • Prototyping: Added interactive transitions between screens
  • Design System: Organized components into a reusable library

Challenges & Solutions

Color Selection Difficulty

Used online color palette generators and accessibility checkers to ensure good contrast and harmony. Tools like Coolors.co and WebAIM's contrast checker were invaluable.

Information Overload

Applied progressive disclosure principles, showing only essential information initially and allowing users to drill down for more details when needed.

Consistency Across Screens

Created a comprehensive style guide and component library to ensure consistency across all app screens and user flows.

Next Steps

The redesign is progressing well, but there's still work to be done:

  • Complete the remaining screen designs (reports, settings, user management)
  • Create detailed prototypes for user testing
  • Collaborate with developers on implementation feasibility
  • Conduct usability testing with actual school administrators
  • Iterate based on feedback and create final design specifications

This project has significantly improved my UI design skills and taught me the importance of systematic design thinking. While color selection remains challenging, I'm developing better instincts for creating visually appealing and functional interfaces.
